# 🚀 AI Study Planner 

### 🎯 AI-Powered Student Performance Prediction System

A modern, premium dashboard that predicts student performance using Machine Learning, provides risk analysis, personalized recommendations, and tracks academic progress — all in a sleek UI.

---

## ✨ Features

* 🔍 **Performance Prediction**
  Predict final score based on academic + behavioral inputs

* 🚨 **Risk Analysis**
  Classifies students into:

  * Low Risk ✅
  * Medium Risk ⚠️
  * High Risk ❌

* 💡 **Smart Recommendations**
  AI-based suggestions to improve performance

* 📈 **What-If Analysis**
  See how study hours impact your score

* 📊 **Feature Importance Visualization**
  Understand what affects performance most

* 🧠 **AI Academic Assistant**
  Ask questions and get improvement tips

* 🔐 **Authentication System**
  Secure login/signup with user history

* 📜 **Prediction History**
  Track past performance predictions

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit (Custom Premium UI + CSS)
* **Backend:** Python
* **Machine Learning:**

  * Random Forest Regressor
  * Scikit-learn
* **Database:** SQLite
* **Visualization:** Matplotlib
* **Data Handling:** Pandas

---

## 🧠 Machine Learning Model

* Algorithm: **Random Forest Regressor**

* Features Used:

  * Study Hours
  * Attendance
  * Sleep Hours
  * Previous Score
  * Social Media Usage
  * Stress Level
  * Preparation Level
  * Extra Activities

* Output:

  * Predicted Final Score
  * Risk Category

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/student-performance-ai.git
cd student-performance-ai
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the app

```bash
streamlit run app.py
```
